CSV is an API for working with csv files.

Key Features

  • Reads multiple CSV formats including comma, tab, semicolon or pipe separated files.
  • Support charset encoding - but in reading and writing
  • Auto detection of CSV format and encoding
  • Export to JSON, XML and HTML
  • Filter your CSV data using a fluid query language.
